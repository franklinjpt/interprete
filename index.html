Interprete de LFL

LFL (LauFrank Language) es un lenguaje de programación interpretado de propósito general, diseñado por Laura Sierra y Franklin Pineda para la asignatura de Compiladores del Programa de ingenieria de sistemas.

Este es un lenguaje facil de usar y con caracteristicas similares a los lenguajes mas reconocidos, el objetivo de crear este lenguaje fue para entender como funcionan los interpretes de lenguaje.

$ nombre = "Franklin";
